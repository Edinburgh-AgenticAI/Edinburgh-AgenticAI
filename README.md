## ğŸ’» Local Development

To run this website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Edinburgh-AgenticAI/Edinburgh-AgenticAI.git
   cd Edinburgh-AgenticAI
   ```

2. Open `index.html` in your browser:
   ```bash
   # On macOS
   open index.html

   xdg-open index.html

   # On Windows
   start index.html
   ```

Or use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js with http-server
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.

## ğŸ”§ Customization

### Updating Content

1. **Hero Section**: Edit the title and subtitle in `index.html` lines 33-44
2. **Research Areas**: Modify research cards in `index.html` lines 87-139
3. **Team Members**: Update team information in `index.html` lines 149-169
4. **Colors**: Change color scheme in `styles.css` CSS variables (lines 12-22)

### Color Scheme

The default color scheme uses:
- Primary: `#6366f1` (Indigo)
- Secondary: `#8b5cf6` (Purple)
- Accent: `#ec4899` (Pink)

To change colors, update the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    --accent-color: #your-color;
}
```
